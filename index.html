    Abstract

    Visual identification of microorganism requires a lot of experience and expertise from a person, who performs identification / recognition, but still canâ€™t guarantee high level of accuracy. Modern high precision approaches (like DNA / special sites sequencing) are expensive and time consuming.

    This project has been designed to be an express tool for visual identification of microalgae cells on picture taken by microscope camera.

    Problem description

    Given:

    Picture of microalgae, made with microscope camera and contained several cells.

    Find:

    • Belonging to specie / genera
    • Probability of correct identification, measured in %

    Approach / Tool:

    Pre-trained Neural Network (NN)

    Deploying and requirements

    Project has been tested on Windows 7 ... 10 (64-bit only) and Ubuntu 14.04 ... 16.04 (64-bit only) with Python 3.5.

    Deploying on Windows:

    Install Python 3.5.2 (other versions were not tested) https://www.python.org/downloads/release/python-352/

    Create the folder for your project (let say D:\MicroAlgae_ID) Open it in explorer and run the command line from current location (click the right mouse button in explorer windows with pressed Shift-key and choose the Run Command Line here from pop-up menu). Install the virtualenv for python (you can install other packages in system python, but it has conflicts in Windows 8-10, so using of virtual environments is strongly recommended) by running:
    pip install virtualenv
    Create the virtual environment:
    virtualenv -p python3 .\
    Enter the created environment:
    .\env\Scripts\activate.bat
    Install other required packages:
    pip install Django TensorFlow Pillow
    Copy the src and tf folders to your project folder (D:/MicroAlgae_ID) and type in opened command line:
    python .\src\manage.py runserver
    To create the one-click start for project create a run.cmd file in project folder with following content:
    call .\env\Scripts\activate.bat
    python .\src\manage.py runserver

    Deploying on Ubuntu:

    Open the terminal and create the project folder (let say ~/ID_MicroAlgae):
    mkdir ~\ID_MicroAlgae
    Install the virtualenv, create the environment and activate it:
    pip install virtualenv
    virtuaenv -p \usr\bin\python3 ~\ID_MicroAlgae\env
    source ~\ID_MicroAlgae\env\bin\activate
    Install all requirements:
    pip install pillow django tensorflow
    Copy the src and tf folders to your project folder (~/ID_MicroAlgae) and type in opened command line:
    python ~\ID_MicroAlgae\src\manage.py runserver
    Now You should have an access to project via browser by entering the address
    127.0.0.1:8080

    Sometimes (if you are deploying on some public free domains like c9.oi ) you could need to modify this command to something like (Please, check the documentation ):
    python ~\ID_MicroAlgae\src\manage.py runserver $IP:$PORT

    To create the one-click start for project create the run.sh file in convenient for you location with following content:
    source ~\ID_MicroAlgae\env\bin\activate && python ~\ID_MicroAlgae\src\manage.py runserver
